# Continuous-Learning-for-Autonomous-Vehicles


Welcome to the autonomous vehicle self-learning project. Here, I demonstrate guiding vehicles to learn from previously unseen environmental elements, enabling accurate motion prediction once these elements are identified and integrated into the vehicle's memory.
